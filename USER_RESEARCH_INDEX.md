## HOW TO USE THIS RESEARCH

### For Feature Ideation
1. Read User Insights > Jobs-to-be-Done
2. Review User Quotes > Desire Quotes
3. Check Prioritization Framework > Kano Model
4. Ideate features that address unmet desires
5. Score with RICE framework

### For Sprint Planning
1. Review Prioritization Framework > Q1 Roadmap
2. Read User Stories > Relevant Epic
3. Refine acceptance criteria with team
4. Create technical tasks
5. Estimate story points

### For Marketing Campaigns
1. Read User Quotes > Pain Point Quotes for empathy
2. Review Executive Summary > Messaging Framework
3. Use User Quotes > Success Stories for testimonials
4. Check Competitive Intelligence for positioning
5. Create content addressing top pains

### For Stakeholder Presentations
1. Start with Executive Summary (10 min read)
2. Pull key data points from User Insights
3. Show User Quotes for credibility
4. Present Prioritization Framework > OKRs
5. Share projected business impact

### For Customer Support
1. Review User Insights > Pains section
2. Read User Quotes > Red Flags (vulnerable users)
3. Prepare responses to common complaints
4. Understand mental health considerations
5. Have resources ready (crisis hotlines)
